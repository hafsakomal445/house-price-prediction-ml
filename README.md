# 🏠 House Price Prediction using Machine Learning

This project predicts house prices using machine learning models such as Linear Regression and Random Forest. It includes data preprocessing, feature engineering, model training, evaluation, and deployment using Streamlit.

---

## 🚀 Live Demo

👉 https://house-price-prediction-ml-v4iqhjtzjjafa6xctus6oj.streamlit.app/

---

## 📊 Project Overview

- Cleaned and preprocessed real-world housing dataset  
- Performed exploratory data analysis (EDA)  
- Engineered meaningful features to improve performance  
- Trained and compared multiple regression models  
- Optimized model using hyperparameter tuning  
- Deployed as an interactive web app  

---

## 🧠 Models Used

- Linear Regression  
- Random Forest Regressor
